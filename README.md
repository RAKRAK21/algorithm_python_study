# algorithm_python_study
재휘, 경훈, 형락의 스터디

# 스터디 방법
  1. 이 저장소를 `fork` 한다.
  2. 생성된 원격 저장소에 본인의 영문 이름으로 폴더를 생성한다.
  3. 생성된 폴더에 자신의 소스코드를 업로드한다.
  4. 원본 저장소로 `pull request`를 한다. (토요일에 merge 예정)
